Paper(s)

A High-Level Modeling Language for Android Framework (MoDroid)

June 2014-Present

In this paper, we discuss our implementation of a high-level modeling language for the Android Framework. Our model is an abstraction of the Android framework that allows the user to create and manipulate Android applications in a way similar to the DOM which allows the user to be more expressive using less code. Our model also proves useful in other areas such as combining separate, independent Android applications into one. I started working on this paper as a research assistant with Dr. Mohammad Jaber at the American University of Beirut on June 2014. The paper is yet to be published. Paper URL: http://goo.gl/7PnYyK

Roomcast

March 2014 - May 2014

Roomcast is an Android social app. Roomcast presents the user after login with a grid and a broadcast button. Clicking the broadcast the button switches the user to another activity that allows the user to upload any type of media to the cloud that get deleted after a predefined period of time. The grid represents the room the user is in, it displays dots that represent all the users in the room, if the user clicks on any of the dots, he/she gets to see what that user broadcasted and the other user gets notified that the user saw what they broadcasted.

Touchtile

May 2013

It was a project I worked on during my sophomore year for the Media Processing elective I was taking. I used a Kinect, a cloth surface enclosed in a frame and code I wrote using the Processing Java framework to create a pressure sensitive touch screen for a more immersive input. Taking advantage of the Kinect's IR emitter, I could scale the touch surface anywhere from 0.25x0.25m to to 2x2m. In the prototype, I created an immersive drawing application. The space of the cloth surface allows the painter to draw as if they were drawing on a real life canvas, and the added pressure sensitivity gives the freedom for the painter to control the stroke of the lines being drawn.
